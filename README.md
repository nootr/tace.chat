## Technical details

- **Encryption**: ChaCha20-Poly1305 with X25519 key exchange
- **Network**: Peer-to-peer with DHT-based routing
- **Privacy**: No logs, no metadata collection, no tracking
- **Compatibility**: Works in any modern browser
